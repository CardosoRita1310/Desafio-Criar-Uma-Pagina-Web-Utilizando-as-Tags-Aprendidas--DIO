Referencia bibliografica

  • HTML

    HTML (abreviação para HyperText Markup Language) é uma linguagem de marcação utilizada para categorizar a estrutura de um documento. Sendo que estas marcações de estrutura são interpretadas por um software de navegação de internet e assim elabora uma página web com os mecanismos de hipermídia interativos ao usuário (BORTOLOSSI, 2012).

  • CSS

    CSS (sigla para Cascading Style Sheets) é uma linguagem de estilo empregada para caracterizar a aparência (cor, fonte e layout) dos diversos itens de um documento que foi criado por uma linguagem de marcação (por exemplo, a linguagem HTML). Ela foi desenvolvida com o objetivo de distinguir a estrutura do documento de sua aparência (BORTOLOSSI, 2012).

  • JavaScript

    O JavaScript é uma linguagem de programação interpretada existente nos navegadores de internet. Sendo assim, esta linguagem possibilita uma série de artifícios de interface gráfica (como exemplo temos campos de entrada, botões e seletores), permitindo assim a geração de páginas web mais interativas. Além disso, a linguagem JavaScript concede ao desenvolvedor alterar e incluir, de modo dinâmico, a aparência e o conteúdo dos diversos elementos que integram o documento (BORTOLOSSI, 2012).
